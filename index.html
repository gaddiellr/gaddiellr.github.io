PROJECTS


Robot Localization

A Puzzlebot was programmed to localize itself in an environment with markers, detecting markers with a camera and obstacles with a LiDAR sensor. The goal was for the robot to move itself to any goal with high precision, without collisions. I was in charge of managing the project and developing the algorithms needed for the obstacle avoidance and a localization filter.


Robot Autonomous Drive

This project was about programming a Puzzlebot to follow a track, responding to traffic signals. This was achived by programming a signal detection neural network in Python, and running it in real-time with the Puzzlebot camera. My job was to create the track detection program in Python, and implementing the control system for following these tracks, as well as leading the project.


Collabotative Bin Picking System

This was a system capable of automatically classifying fruit and placing it in containers. A pneumatic soft gripper system was developed and attached to the Ufactory Lite6 robot, along with a camera running a neural network. My specific tasks were developing a visual interface with the real-time system data, making a calibration for the camera, creating the state machine robot program and a database storage program. I also set up the system components physically, and connected them with the control computer via MQTT.


Drone Signal Tracking

A DJITello Drone was controlled with hand signals made at a camera. A Python program opened the camera, detected hand keypoints using the MediaPipe library, then made the sign detection and sent instructions to the drone. I worked in creating the functions for classifying the hand keypoints, and the drone response coding.


Temperature Controller

This was a PID Control System implementation, using a REX-C100 Controller, with the goal of keeping a constant temperature of 20°C for a container with 20 liters of water to be used in a metrology lab. I made some experiments for this project, collecting the data for analysis. I used Matlab for analyzing the system efficiency. I also lead this project and assigned tasks.
